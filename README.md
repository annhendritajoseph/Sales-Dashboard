# Sales Dashboard (Power BI)

## Overview
This project presents a Sales Dashboard built using Power BI. It analyzes sales, profit, cost, and regional performance.

## Features
- Sales & Profit Analysis
- Category & Buyer Insights
- Time-based trends using Date table
- Regional performance breakdown

## Data Model
The project follows a relational data model where:
- Sales is the fact table
- Buyers, Managers, Regions, and Dates are dimension tables
